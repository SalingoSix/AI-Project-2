Project by Alex Medos

Project runs in Debug x64 mode

Controls:

WASD - Controls the camera left and right, zooms in and out

IJKL - Controls the player character

J and L will rotate the character around,
I and K will move him forward and backward in the direction he's facing

ESC to exit

At 20 health, the angry opponents will begin to charge the player.
If an opponent type is missing, try reloading as they are randomly generated.

Adjust parameters in the config file found in the assets folder.
